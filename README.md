# MoyJS

MoyJS 前端框架全产品仓库

## 目录说明

```
docs 框架文档的md源文件
packages 框架的核心模块、核心组件、解决方案等包
scripts 框架的构建产出、文档整合等执行脚本
website 框架的官网文档站点
```
